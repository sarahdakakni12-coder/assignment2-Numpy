This project contains a Jupyter Notebook that demonstrates the use of NumPy, a powerful Python library for numerical computing. The notebook includes examples, exercises, and solutions related to array operations and basic data manipulation.

File Included
firstassigment3040 (3).ipynb – Main notebook with all code and explanations.
Objectives

The purpose of this assignment is to:

Understand NumPy arrays and their properties
Perform mathematical operations on arrays
Practice indexing and slicing
Explore basic data analysis techniques using NumPy
Requirements

To run this notebook, you need:

Python 3.x
Jupyter Notebook or JupyterLab
NumPy library

You can install NumPy using:

pip install numpy
How to Run
Download or clone the repository
Open the notebook using Jupyter Notebook or JupyterLab
Run the cells in order to see the outputs
Topics Covered
Creating NumPy arrays
Array indexing and slicing
Mathematical operations
Reshaping arrays
Basic statistics with NumPy
